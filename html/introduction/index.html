introduction to HTML

HTML is a markup language consisting of a series of elements used to wrap (or enclose) text content to define its structure and cause it to behave in a certain way.

the difference in structure between HTML4 & HTML5

there are two types of tags in HTML

  1. empty tags:

    Empty tags, also known as self-closing tags, do not have any content between an opening and closing tag because they do not enclose any text or other elements.

    • <img>: Used to embed images.
    • <br> : Inserts a line break.
    • <hr> : Represents a thematic break (horizontal rule).
    • <input>: Represents an input field in forms.

  2. container tags:

    Container tags enclose content, which can include text, other HTML elements, or both. They consist of an opening tag and a corresponding closing tag.

    • <div>: A generic container for block-level elements.
    • <p>: Represents a paragraph of text.
    • <h1> to <h6>: Header tags for different levels of headings.
    • <a>: Represents a hyperlink.

there are two types of element

type Description
inline These elements are on a single line unless separated by a space between them.
block These elements are in blocks and leave spaces before and after them, so that each element is on a separate line.
