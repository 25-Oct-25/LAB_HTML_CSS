ParkPro – Smart Parking Application

Graduation Project (2025)

ParkPro is an innovative smart parking system designed to reduce traffic congestion and improve urban mobility. The project integrates Flutter, Firebase, and Arduino to enable real-time parking space monitoring, reservation, and control through a secure mobile app.

I implemented secure data flow between sensors and the app using Firebase Realtime Database, ensuring accurate availability tracking and remote gate control. The system also includes security measures such as authentication, logging, and access rules to prevent unauthorized use.

ParkPro aims to support Saudi Arabia’s Vision 2030 by enhancing smart city infrastructure, sustainability, and digital transformation.

Library Management System

Desktop Application (2024)

A management system developed using C# and SQL Server that simplifies book cataloging, user registration, and borrowing operations for small-scale libraries.

The project follows Object-Oriented Programming (OOP) principles, allowing reusable and organized code structure for future scalability. It features an intuitive interface, secure database interactions, and efficient search and update functionalities.

This project strengthened my backend logic design and database management skills while maintaining user-friendly software design.

Card Store App

Mobile Application (2023)

A simple e-commerce mobile application developed using Swift and SwiftUI, designed for selling greeting cards and digital gift items with customizable options.

The app features product listing, cart management, and adjustable settings for personalization. Its design focuses on minimalism, responsiveness, and smooth user navigation.

This project was my introduction to iOS development and mobile UI design, combining creativity with functionality to enhance user experience.
