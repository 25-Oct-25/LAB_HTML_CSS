CSS Box Model

Every element is a box — learn its structure.

Structure

The box model includes: Content → Padding → Border → Margin.

Box Example
.box {
  width: 200px;
  margin: 20px;
  padding: 20px;
  border: 5px solid blue;
}

Box-Sizing

By default, box-sizing is content-box. Use border-box to include padding and border inside the width. 

* { box-sizing: border-box; }

Margin Collapse

Vertical margins between elements may collapse. Use padding or borders on the parent to avoid unexpected spacing.
